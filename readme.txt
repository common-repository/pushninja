== Installation ==
Installing PushNinja WordPress plugin is easy. Head to the WordPress.Org plugin directory and search for the plugin to start your journey of improving website performance. 

Here’s a 3 step guide to help you install and activate PushNinja plugin from your WordPress dashboard.

1.Search and Install [PushNinja](https://wordpress.org/plugins/pushninja/) from the WordPress.org plugin directory 
2.Alternatively, You can upload the PushNinja plugin folder to your wp-content/plugins directory
3.Activate PushNinja plugin from your dashboard.
4.Get started with PushNinja

After Installation, see a PushNinja wizard in your dashboad and start engaging your website visitors with push notifications.
